Sudoku Solver

For a long time Ive tried to make an efficient Sudoku solver in Python, I tried avoiding using a brute force backtracking algorithm for a long time worrying about the speed and performance.
After my C++ final exam, I immedietly went home and made a backtracking algorithm in C++ in a few hours and its fast
You just simply enter the board data on a txt file and the program will read it, solve the board and display the results
Im planning on making it automated to read a board directly from Sudoku.com and solving it automatically on the website. Just need to get good at web dev and web scraping
